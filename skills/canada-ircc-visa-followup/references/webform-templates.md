# IRCC Web Form Templates

## Delayed Visitor Visa Inquiry

```text
Dear IRCC Officer,

I am writing to inquire about my visitor visa / temporary resident visa application.

My application was submitted online on [SUBMISSION_DATE], and my IRCC account currently shows that the application is still in progress. As of [TODAY_DATE], it has been approximately [ELAPSED_DAYS] days since submission. The current IRCC processing time shown for [COUNTRY_OR_REGION] visitor visa applications is [PROCESSING_TIME], which my application has significantly exceeded.

There are no new messages, document requests, interview requests, or medical exam requests in my IRCC account. Could you please confirm whether my application is still active and whether any additional information or action is required from me?

Thank you for your assistance.

Sincerely,
[FULL_NAME]
```

## Short Status Inquiry

```text
Dear IRCC Officer,

I would like to ask about the status of my [APPLICATION_TYPE] application submitted on [SUBMISSION_DATE]. My online account still shows the application as in progress, and I have not received any request letter or new message.

Could you please confirm whether the application is still being processed and whether I need to provide any additional information?

Thank you.

Sincerely,
[FULL_NAME]
```

## Chinese Summary For User Review

```text
英文说明的大意是：你在 [SUBMISSION_DATE] 在线提交了 [APPLICATION_TYPE]，截至 [TODAY_DATE] 已等待约 [ELAPSED_DAYS] 天，超过当前 IRCC 官方显示的 [PROCESSING_TIME] 处理时间。账号中没有补料、面试、体检或新消息，因此请求 IRCC 确认申请是否仍有效、是否还在处理中，以及是否需要你补充材料或采取行动。
```

## Field Checklist

- Applicant role: principal applicant
- Application category: Temporary Residence
- Application type: Temporary Resident Visa for visitor visa cases
- Submission method: Online, if confirmed from account
- Name: exactly as shown in the IRCC account/passport
- Email: user-confirmed email, typed manually if confirmation field blocks paste
- Date of birth: match IRCC/passport format
- UCI: read from IRCC account or provided by user
- Application number: starts with a letter for many visa applications, e.g. `V...`
- Passport number: ask user if required; do not infer
- Consent/disclaimer: summarize and get confirmation before checking
